## Key Design Decisions

1. **Snapshot Storage**: Compressed JSON + Parquet for efficiency
2. **Validation**: Pre-flight checks before replay execution
3. **Transparency**: Zero code changes needed in most modules
4. **Safety**: Snapshots only in live mode, replay-only loads
5. **Backwards Compatible**: Live mode unaffected, existing replays enhanced
